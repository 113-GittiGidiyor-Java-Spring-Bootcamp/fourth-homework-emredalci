# Show Students

Represent all students

**URL** : `/api/students/find-all`

**Method** : `GET`

---

## Success Response

**Code** : `200 OK`

**Sample Response** :

```json
[
  {
    "name": "Ahmet",
    "birthDate": "2000-01-01",
    "address": "Istanbul",
    "gender": "MALE",
    "id": 1
  }
]
```
